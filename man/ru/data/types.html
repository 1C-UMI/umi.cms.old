Тип данных - это набор характеристик, определяющих назначение объекта. Например, Тип данных Пользователь содержит такие характеристики как Логин, Пароль, E-mail и другие, целью которых служит идентификация конкретного Пользователя, будь то регистрация в форуме или интернет-магазине. Есть базовый набор характеристик, который вы можете расширить с помощью этого модуля, создав, например, дополнительное поле Резюме, с помощью которого пользователь может загрузить на сервер текстовый файл.

Определённый тип данных может содержать в себе подтипы. Нажмите на ссылку Раздел сайта.

Откроется еще один список с типами, содержащими индивидуальные наборы характеристик. Но тем не менее они объединены в один тип - Раздел сайта по схожести большинства признаков и целей. Ведь будь то раздел каталога или публикация - всё это так или иначе страницы вашего сайта.
