Справка

Название поля — название поля, отображаемого при добавлении/редактировании объекта сайта.

Идентификатор — при помощи него мы можем вывести значение поля на сайт. Работает идентификатор точно так же как и макрос. Т.е., чтобы значение поля появилось на сайте, необходимо записать его в виде: %identifikator% и вставить либо в шаблон либо на страницу сайта. Идентификаторы можно также использовать в качестве параметров для других макросов.

Тип поля — позволяет выбрать необходимый тип поля из выпадающего списка

Использовать справочник — справочник используется только вех видах полей, где присутствует список. Таких типов два: выпадающий список и список с множественным выбором. Читайте о создании и назначении справочника в этих топиках.

Видимое — при включённом флажке поле будет отображаться в свойствах объекта. Соответственно при желании поле можно временно отключить, сняв флажок.

Наследуемое — этот флажок очень полезен, когда вы создаёте иерархию типов данных и хотите, чтобы часть полей автоматически попала в дочерний тип данных. Например, вы занимаетесь продажей телевизоров, и в вашем магазине есть разные типы телевизоров: ЭЛТ, ЖК, плазменные. Все они имеют некоторые общие свойства: диагональ экрана, энергопотребление, срок гарантии. А некоторые свойства индивидуальны для каждого типа: параметр "прогрессивная развёртка" свойственнен исключительно жк-телевизорам. Поэтому при создании типа данных "телевизоры" имеет смысл пометить общие свойства как "наследуемые". Тогда затем при создании дочерних типов данных, содержащих индивидуальный набор параметров для каждого из типов телевизоров, общие свойства будут автоматически включены в каждый дочерний тип.

Индексировать — при включённом флажке значение поля будет индексироваться поисковой системой по сайту.

Использовать в фильтрах — при включённом флажке посетитель сайта будет иметь возможность искать объект, используя фильтрацию по этому полю.
