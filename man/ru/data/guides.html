Справка

Справочник — это тип данных, который объединяет объекты этого типа в один раздел. Введены для создания выпадающих списков.

Содержания справочника — это объекты справочника (элементы выпадающего списка).

Обычно справочники используются для того, чтобы задать несколько значений (отображаться на сайте будут, например, через запятую).

Пример: хотим, чтобы у товара было свойство "Поддерживаемые форматы аудио". Решение: создаем справочник "Форматы аудио", заносим в него: "mp3", "wav", ... . Создаем у типа товара поле, которое использует в качестве справочник "Форматы аудио", и выбираем нужные значения.

Ссылка с названия справочника открывает его содержимое. Эти элементы будут представлены в выпадающем списке:

В большинстве случаев используется одно поле и значение, однако, можно вводить дополнительные атрибуты, например, для вывода пиктограмм.
