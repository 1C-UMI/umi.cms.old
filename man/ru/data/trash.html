В "Мусорную корзину" попадают любые удалённые страницы сайта, которые при желании можно восстановить. В таблице указан раздел, из которого был удалён файл, и дата удаления.

Чтобы восстановиить файл из корзины, нажмите на значок "Восстановить" напротив соответствующего файла. Файл вновь появится в том разделе, из которого был удалён.

Вы можете очищать "Корзину", удаляя файлы из неё по одиночке. Для этого нажмите на значок "Удалить" напротив файла. Сверху таблицы есть ссылка "Очистить корзину". Нажатие на неё приведет к полному удалению всех находящихся в ней файлов. Любое удаление файлов их "Корзины" приведёт к невозможности их восстановления в дальнейшем.
