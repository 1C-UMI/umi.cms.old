Главное окно модуля предстваляет собой таблицу, логически разделённую на две части двойной линией посередине. Верхняя часть таблицы содержит список созданных групп. В скобках рядом с именем группы указано число пользователей, входящих в эту группу. Нижняя часть таблицы содержит две ссылки: Пользователи вне групп и Все пользователи. Нажав на эти ссылки вы попадёте в соответствующий список пользователей.

Группы пользователей служат для объединения определенного набора прав и делегирования их отдельным Пользователям. Когда Вы включаете Пользователя в группу, он автоматически получает все права, активированные для данной группы. Если вы изменяете набор прав для данной группы, права всех пользователей, входящих в эту группу, изменяются соответственно.
