Логин и Пароль. Новый Пользователь будет использовать эту учётную запись для доступа в систему. Поля являются обязательными для заполнения.

Если Вы уже создавали группы Пользователей раньше, то можете сразу внести нового Пользователя в одну или несколько этих групп. Для этого поставьте галочку напротив соответствующего названия группы.

Кнопка-флажок Активен по умолчанию включена. Если Вы не хотите, чтобы данный Пользователь мог немедленно пользоваться системой после его создания, снимите галочку. В таком случае Пользователь будет создан, но его учётная запись будет блокирована. После создания Пользователя Вы можете активировать его учётную запись в любой момент.

По необходимости Вы можете заполнить поля с личными данными Пользователя: Фамилия, Имя, Отчество, Пол, Возраст, Аватара.

В секции Настройки прав доступа отметьте галочками необходимые для данного Пользователя права. В случае, если Вы включили его в уже созданную группу, галочки, соответствующие выбранным для данной группы правам, расставятся автоматически. Тем не менее, Вы можете изменить набор прав, поставив дополнительные или убрав нужные галочки.

Вы можете самостоятельно расширять и редактировать набор полей, необходимых для заполнения при создании/редактировании Пользователя. Для этих целей существует модуль Шаблоны данных. Читайте документацию по нему в соответствующем главе руководства.
