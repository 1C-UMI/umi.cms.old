Название баннера будет показано в списке баннеров.

Тип баннера (картинка, Flash или HTML-код).

URL (адрес страницы, которая откроется при нажатии на баннер).

Открыть в новом окне — страница, к которой ведет рекламный баннер, откроется в новом окне.

Комментарии — это короткий пояснительный текст, предназначенный для администраторов. Выводится в списке баннеров в качестве примечания.

Активен. Если она отмечена, то баннер показывается на сайте. Чтобы приостановить показ баннера, нужно убрать эту галку и нажать кнопку «Сохранить».

Показывать на страницах с тегами — данный баннер будет показан только на тех страницах, которые помечены соответствующими тегами.

Показывать пользователям с тегами — данный баннер будет показан пользователям, "собравшим" соответствующие теги.

Подробнее о работе тегов читайте в топике Как пользоваться тегами.

Количество показов. Показывает текущую статистику по показам данного баннера на сайте.

Максимальное количество показов. Введите сюда число, если вы хотите автоматически отключить баннер после определенного количества показов.

Количество переходов. Показывает число переходов с данного рекласного баннера на страницу рекламодателя на текущий момент.

Дата начала и окончания показа. Позволяет сначала создать баннер, а потом автоматически включить и отключить его в заданную дату. Датой начала показа автоматически ставится дата создания баннера, это число можно изменить. Дата окончания показа исходно не задана, то есть баннер будет показываться без ограничения по срокам.

В Разделе отобраения можно выбрать те разделы сайта, в которых баннер будет появляться. Для этого переместите при помощи кнопки со стрелкой в правую часть названия тех страниц сайта, на которых баннер будет размещён.

В поле Место показа выделите название той зоны на странице, где баннер должен появляться.

Временной таргетинг предназначен для того, чтобы показывать баннер на сайте только в определенное время. Модуль позволяет показывать баннеры: По числам месяца, По месяцам, По дням недели, По времени суток.
