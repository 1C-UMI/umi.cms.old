Нажатие на название сообщения в списке откроет страницу редактирования свойств сообщения:

Название — название рассылки, которое будет отображаться в списке рассылок в администраторской панели.

Заголовок — заголовок сообщения будет отображаться в письмах пользователям.

Внизу расположено ещё одно поле, которое можно использовать для прикрепления файла к письму рассылки.
