На этой вкладке администратор может увидеть список рассылок, поддерживаемых на сайте.

При нажатии на название рассылки открывается список сообщений, входящих в эту рассылку. Справа расположены пиктограммы для управления рассылками. Они позволяют изменять настройки, смотреть список пользователей, подписанных на определенную рассылку, а также удалить рассылку.

Над списком находится ссылка для добавления новой рассылки. При нажатии на нее откроется окно с настройками новой рассылки.
