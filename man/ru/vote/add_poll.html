Название опроса будет показано в списке голосований, в пути к странице. Поля title, ключевые слова и описания сделаны для поисковой оптимизации сайта. Подробнее об этом читайте в топике Редактирование мета-информации раздела Редактирование страниц.

Поле Текст вопроса предназначено для того, чтобы подробнее описать посетителям сайта вопрос.

Опрос активен. Если отмечена эта галка, посетители сайта будут видеть опрос на сайте. По умолчанию опрос активен.

Голосование закрыто. Если отмечена эта галка, то опрос становится закрытым, то есть посетители не смогут голосовать, но по-прежнему смогут видеть результаты опроса.

Конечно же, в каждом опросе необходимо дать посетителям варианты ответов. Для этого предусмотрена специальная область в свойствах опроса:

Чтобы добавить вариант ответа, предусмотрена нижняя пустая строчка. Введите туда текст ответа и нажмите кнопку Сохранить. После этого появится вариант ответа. Аналогичным образом можно изменить текст других ответов и сохранить изменения.
