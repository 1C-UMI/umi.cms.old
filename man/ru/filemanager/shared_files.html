Справка

На закладке Доступные для скачивания файлы вы можете загрузить файлы, которые посетители сайта могут скачать.

В списке показаны файлы уже доступные для скачивания. Здесь вы можете редактировать файл, удалить либо сделать недоступным для скачивания.

Чтобы добавить файл, доступный для скачивания, нажмите на ссылку Добавить файл
