Справка

Данный модуль предназначен для быстрого доступа к файлам и папкам вашего сервера. Вы можете создавать новые директории, а затем помещать туда файлы, которые посетители сайта могут скачивать.

При заходе в модуль, вы попадаете автоматически в директорию /files, предназначенную для размещения там файлов. Для закачки файлов непосредственно в эту директорию внизу есть область Закачать файл, с помощью которой вы можете загрузить на сервер любой файл с вашего локального компьютера.

Вы также имеете возможность создать свою иерархию директорий как внутри папки /files, так и внутри любой дргуой директории. Для этого вверху закладки "Файловая система" есть ссылка Создать директорию.
