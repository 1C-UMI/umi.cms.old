Нажатие на ссылку с названия объекта откроет страницу редактирования его свойств. Страница довольно большая, состоит из нескольких блоков. Большая часть из них вам уже знакома, если раньше вам доводилось создавать какие-либо объекты или страницы в сиситеме UMI.CMS. Описание этих свойств можно почитать в разделе Редактирование страниц.

В свойствах товара есть область Производитель, куда вы можете вписывать названия компаний, а затем выбирать из списка. Для этого под списком производителей есть доплнительная строка. Чтобы добавить производителя, впишите его название в эту строку и сохраните страницу.

Один товар может быть помещен в несколько категорий для удобства посетителей сайта. Например, детские сиденья для автомобилей можно поместить как в автомобильные товары, так и в товары для детей. Разные люди будут искать по-разному. Для этого создайте виртуальную копию страницы с товаром из дерева сайта. Подробнее об этом читайте в разделе Редактирование структуры сайта.

В разделе Сопутствующие товары вы можете выбрать товары из каталога, которые, по вашему мнению, могут быть использованы вместе. Например, гарнитура и мобильный телефон.
